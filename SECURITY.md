# Security Policy

## Supported Versions

PipeSort is actively maintained on the latest code in the `main` branch.

| Version / Branch | Supported |
| ---------------- | --------- |
| `main`           | ✅ Yes    |
| Older releases   | ❌ No     |
| Experimental branches (`test/*`, `experiment/*`) | ❌ No |

Security fixes are applied only to the current `main` branch.

## Reporting a Vulnerability

If you discover a security issue, please **do not open a public issue**.

Instead:
- Use **GitHub Security Advisories**, or
- Contact the maintainer privately (if a contact method is listed)

When reporting, include:
- A clear description of the vulnerability
- Steps to reproduce (if possible)
- Potential impact
- Any suggested fix (optional)

## Response Process

- You can expect an acknowledgment within a reasonable time.
- Valid issues will be investigated and fixed as soon as possible.
- Once resolved, a patch or update will be released on `main`.

Thank you for helping keep PipeSort safe.
